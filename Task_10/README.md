# Task 10 - Firewall Rules Documentation

## Objective
To understand firewall concepts and configure rules to control network traffic securely.

## What I Did
- Learned basic firewall concepts and functions.
- Configured firewall rules to allow and deny network traffic.
- Allowed required ports and blocked unnecessary ports.
- Tested network connectivity after applying rules.
- Reviewed firewall logs to monitor traffic.
- Blocked malicious IP addresses.
- Documented firewall rules and analyzed their impact.

## Key Learnings
- How firewall rules protect systems from unauthorized access.
- Importance of allowing only necessary services.
- Role of firewall logs in security monitoring.

## Tools Used
- UFW / Windows Firewall
- Local system and network

## Outcome
This task improved my understanding of firewall configuration and network security controls.
